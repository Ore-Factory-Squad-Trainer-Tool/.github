### Quick Start Instructions

1. Extract the trainer files.
2. Run OreFactorySquadTrainer.exe as Administrator.
3. Launch Ore Factory Squad and load your factory.
4. Press F12 to attach the trainer.
5. Open control panel with INSERT.
6. Enable core modules for mining or building.
7. Adjust multipliers and test in a small operation.
8. Detach with F11 before closing the game.

### Risk & Stability Notes
